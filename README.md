# Super Simple GraphQL Tutorial with C#

This is the companion repository for the [blog post](https://developer.okta.com/blog/2020/08/24/simple-graphql-csharp) on Okta's developer blog.

## Getting Started

Clone the repository by:

```sh
git clone https://github.com/oktadeveloper/OktaGraphQL.git
```

## Get Help

If you have any questions, you can post them on [the blog post](https://developer.okta.com/blog/2020/08/24/simple-graphql-csharp).

If you have questions regarding developing with Okta, you can post them on the [developer forum](https://devforum.okta.com/).
